# E-Commerce Sales Analysis

End-to-end data analysis project on Brazilian e-commerce data (Olist dataset) using Python and Streamlit.

## Live Dashboard
https://ecommerce-sales-analysis-srv.streamlit.app/

## Key Insights
1. Revenue grew steadily from 2016 to 2018 indicating strong business scaling
2. Bed/bath, health/beauty and computer accessories are the top 3 revenue categories
3. Average order value (R$204) is almost double the median (R$109) suggesting high value orders skew the average
4. São Paulo, Rio de Janeiro and Minas Gerais account for majority of orders
5. Monday is the peak ordering day suggesting customers plan purchases start of week

## Tech Stack
Python, Pandas, Plotly, Streamlit

## Dataset
Olist Brazilian E-Commerce Public Dataset via Kaggle
